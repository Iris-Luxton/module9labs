# Programming task - REST API using MySQL
## Technical details
The project will be executed with the following command:
```sh
npm install express cors sequelize
```
To run/test your code, run this command:
```sh
node index.js
```
For example:
```js
[
    {
        name: 'Sebastian Webster',
```
